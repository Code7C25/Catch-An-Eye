***BUGS ENCONTRADOS:***



* **Scroll únicamente hacia abajo** --> **CORRECIÓN**: Se corrigió una línea de código que impedía el movimiento hacia arriba
* **Scroll poco sensible a movimientos** --> **CORRECIÓN**: Se ajustaron los parámetros de sensibilidad del movimiento



* **Guiños invertidos** --> **CORRECIÓN**: Se corrigieron varias líneas de código que intercambiaban los guiños involuntariamente



* **Calibración en segundo plano** --> **CORRECIÓN**: Se implementaron nuevas líneas de código que intercambiaban los guiños involuntariamente



* **Poca o nula precisión al momento de mover el cursor** --> **CORRECIÓN**: Se añadieron nuevos filtros y cálculos que permiten mejorar mucho más la precisión cuando se enfoca en un punto específico
